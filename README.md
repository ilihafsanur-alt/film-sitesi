# 🎬 Movie Universe (Film Dünyası)

A dynamic, interactive, and fully responsive web application built with HTML, CSS, and JavaScript. This project allows users to browse a curated list of movies, filter them by genre, and interact with the application by leaving comments and ratings for each movie.

## ✨ Features

- **Dynamic UI Rendering:** Movie cards are generated dynamically using JavaScript based on a structured data array.
- **Genre Filtering:** Users can filter the movie grid by categories (Drama, Fantasy, Sci-Fi, Horror, Animation).
- **Interactive Comment System:** Each movie features an individual comment section where users can input their name, leave a rating (1-10), and write a review.
- **Responsive Grid Layout:** Built with modern CSS Grid (`auto-fit`, `minmax`) to ensure a seamless viewing experience across all devices (mobile, tablet, and desktop).
- **Modern Styling:** Features hover animations, custom form styling, and a clean, user-friendly color palette.

## 🛠️ Technologies Used

- **HTML5:** Semantic structure and custom form validations.
- **CSS3:** Flexbox, CSS Grid, custom properties, and smooth transitions.
- **JavaScript (ES6+):** DOM manipulation, template literals, array iterations (`forEach`), and event handling.

## 📂 Project Structure

- `index.html`: The main entry point containing the semantic structure and filter dropdown.
- `style.css`: Contains all the styling, including the responsive grid and interactive elements.
- `script.js`: Houses the movie data array and the logic for rendering the UI and handling user interactions.
